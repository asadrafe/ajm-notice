# AJM Notice Generator

Official notice generator for A.J.M. Academy, Kishanganj, Bihar.

## Features
- 5 notice templates (General, Holiday, PTM, Exam, Fee Reminder)
- Issued by: School Management / Director / Principal (In-Charge)
- Upload school logo (replaces placeholder seal)
- Upload signatures for Director & Principal
- Download as PDF or JPG (black & white, print-ready)
- Hindi + English body support

## Deploy to Vercel (via GitHub)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import the GitHub repo
4. Vercel auto-detects React → click **Deploy**
5. Done ✅

## Run Locally

```bash
npm install
npm start
```
